# AndroidInfo

An assistant for developers to visualize the info of Android.

### Include the following the info of Android
- [x] OS build info
- [x] Telephony info
- [x] Display info
- [x] PackageInfo
- [x] ApplicationInfo
- [x] ActivityInfo, ProviderInfo, ServiceInfo
- [x] ConfigurationInfo
- [x] InstrumentationInfo
- [x] Storage info
- [x] NetworkInfo
- [x] MemoryInfo
